# Bayesian Hierarchical Model of Individual Player Performance

This project estimates player abilities during the FIFA World Cup 2022 using Bayesian hierarchical models.

Models included:
- Passing model
- Shots model
- Interceptions model

Methods:
- Poisson and Negative Binomial models
- MCMC estimation using JAGS

## Data Source

The dataset used in this project was obtained from Kaggle and contains player performance statistics from the FIFA World Cup 2022. Link: https://www.kaggle.com/datasets/swaptr/fifa-world-cup-2022-player-data?resource=download
